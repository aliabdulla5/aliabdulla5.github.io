# GraphQL Profile Dashboard

A Super Mario-themed web application that fetches and visualizes user profile data from a GraphQL API. The application provides an interactive dashboard with authentication, multiple data views, and dynamic SVG-based statistics.

**Live Demo**: [https://aliabdulla5.github.io](https://aliabdulla5.github.io)

---

## Core Features

- **Authentication**: Secure JWT-based login and logout system
- **Profile Sections**: User information, statistics, XP progress, and skills analysis
- **Data Visualization**: Four SVG graphs including level circle, projects timeline, and dual skills radar charts
- **Responsive Design**: Fully responsive interface optimized for desktop, tablet, and mobile devices
- **Three-World Navigation**: Mario-themed interface with Overworld, Clouds, and Underground sections

---

## GraphQL Requirements

This project implements all required GraphQL query types:

- **Normal Queries**: Simple data retrieval without arguments
- **Nested Queries**: Queries accessing nested object fields and relationships
- **Queries with Arguments**: Filtering using `where` clauses, `_eq` and `_like` operators, and `order_by` sorting

The application connects to the GraphQL endpoint at `learn.reboot01.com` and uses GraphiQL-compatible queries for all data operations.
